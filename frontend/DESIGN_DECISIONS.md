# Design Decisions

## Current Palette

- Background: `#FCF9EA`
- Text: `#1A1A1A`
- Links: `#6f4a86`
- Visited links: `#50365f`
- Democrats: `#5EABD6`
- Republicans: `#E14434`
- Table headers: `#e3d8bd`

## Typography

- Body copy keeps the site's plain document style with `Times New Roman`.
- Tables use the local system monospace stack so numeric columns and rows scan cleanly without the thin, typewriter-like feel of Courier New.
- Table headers are bold.

## Interaction

- Tables are sortable and filterable with a small vanilla JavaScript helper.
- Table filtering uses one row-wide filter box on data-heavy pages.
- The site avoids heavy JavaScript frameworks and keeps navigation as normal static links.
